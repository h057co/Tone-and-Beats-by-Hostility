# Assets - Recursos Gráficos

Este proyecto usa los siguientes recursos gráficos:

## Imágenes

- `HOST_BLANCO.png` - Logo blanco para temas Dark y Blue
- `HOST_NEGRO.png` - Logo negro para tema Light

## Icono

- `HOSTBLANCO.ico` - Icono de la aplicación

## Uso

Los logos se cargan dinámicamente según el tema seleccionado:
- Tema Dark: HOST_BLANCO.png
- Tema Blue: HOST_BLANCO.png
- Tema Light: HOST_NEGRO.png
